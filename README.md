# 📚 Library Management System (Java)

A file-based Library Management System built using Java that allows users to manage books efficiently.

## 🚀 Features
- Add Book
- View Books
- Issue Book
- Return Book
- Search Book
- Data stored in file (library.txt)

## 🛠 Tech Stack
- Java
- OOP Concepts
- File Handling
- ArrayList

## ▶️ How to Run
javac LibrarySystem.java  
java LibrarySystem  

## 💡 Key Concepts Used
- Object-Oriented Programming (OOP)
- File Handling (BufferedReader, BufferedWriter)
- Data Structures (ArrayList)
